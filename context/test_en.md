## Features

This template supports:

- **Full Markdown syntax**
- **Syntax-highlighted code**
- **Auto-generated table of contents**
- **Responsive layout**
- **Light / dark themes**
- **Bilingual articles via `*_zh.md` / `*_en.md`**
